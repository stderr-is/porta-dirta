### 1. Nginx Security Headers (CRITICAL)
**Why:** The `_headers` file only works on Cloudflare Pages. The live VPS nginx has **zero security headers** ‚Äî `curl -I` returns only `content-type` and `Date`.

**File:** VPS `/etc/nginx/sites-enabled/*` (portadirta.com server block)

**Add inside the `server` block:**
```nginx
# Security headers
add_header X-Frame-Options "DENY" always;
add_header X-Content-Type-Options "nosniff" always;
add_header Referrer-Policy "strict-origin-when-cross-origin" always;
add_header Permissions-Policy "camera=(), microphone=(), geolocation=(), payment=()" always;
add_header Strict-Transport-Security "max-age=31536000; includeSubDomains; preload" always;
add_header Content-Security-Policy "default-src 'self'; script-src 'self' 'unsafe-inline' https://fonts.googleapis.com; style-src 'self' 'unsafe-inline' https://fonts.googleapis.com https://fonts.gstatic.com; font-src 'self' https://fonts.gstatic.com; img-src 'self' data: https://maps.gstatic.com https://maps.googleapis.com; frame-src https://beds24.com https://www.beds24.com https://maps.google.com https://www.google.com; connect-src 'self' https://restaurante.portadirta.com https://eventos.portadirta.com https://beds24.com https://maps.googleapis.com; object-src 'none'; base-uri 'self'" always;
```

### 2. Nginx Cache Rules
**Why:** No `Cache-Control` headers on static assets. Browsers re-download everything on every visit.

**File:** VPS `/etc/nginx/sites-enabled/*` (portadirta.com server block)

**Add before the main `location /` block:**
```nginx
# Astro hashed assets ‚Äî immutable, 1 year
location /_astro/ {
    proxy_pass http://localhost:4321;
    proxy_set_header Host $host;
    add_header Cache-Control "public, max-age=31536000, immutable";
}

# Images, video, logos ‚Äî 30 days
location /assets/ {
    proxy_pass http://localhost:4321;
    proxy_set_header Host $host;
    add_header Cache-Control "public, max-age=2592000";
}
```

- [ ] Add cache location blocks
- [ ] Verify: `curl -sI https://www.portadirta.com/_astro/someFile.js` shows `Cache-Control: public, max-age=31536000, immutable`
